Chaos Project

This Computer Science Project was probably the most fun one in the three years I have been here. It called for each of us to create a Visual Basic program that was just all over the place. We could create literally anything we wanted, we just need a certain number of buttons, text boxes, etc. I created a "passive" project, which basically looks very calm on the start screen, but is very chaotic when you click on each of the buttons.
