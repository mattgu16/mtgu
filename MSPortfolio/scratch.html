Scratch Project - Maze Game

The Scratch Project was another group project. The project was for the Computer Science class, and we were to create an elaborate program on Scratch, whether it be a game or something else. I had taken a Scratch class in the summer, so I used one of my old games that I made, and made more levels and fixed old ones. The game's last level was very difficult, and everyone relentlessly tried to beat it.
